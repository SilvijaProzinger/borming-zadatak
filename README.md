# Borming zadatak

Ovaj projekt je kopija homepagea stranice https://www.lovezagreb.hr, napravljen uz pomoć Reacta i initial call-a https://www.lovezagreb.hr/page. 

## Features:

Budući da je cilj zadatka bilo napraviti brzinsku kopiju pokušala sam prikazati sve ključne elemente homepagea koji se mogu pronaći u lovezagreb/page i prekopirati većinu stylinga uz pomoć konzole s originalne stranice. Dodatna svojstva koja bi ubuduće bilo još dobro dodati da stranica bude idealna kopija bila bi: 
* slider efekt za Latest images
* slider efekt i pozadinske slike za Explore topics
* background gradient i zoom efekt za priče

## How to run:

* Downloadajte ili klonirajte repozitorij
* Instalirajte sve dependencije sa npm install
* Pokrenite lokalni server sa `npm start`
* Pokrenite na http://localhost:3000

## Code dependencies:
* create-react-app
* lovezagreb.hr/page
